
# 💻 Practica 1 + 2 Seminario (Python)

Practicas 1 y 2 de Seminario de Lenguajes (opcion Python) UNLP.

A continuacion encontraras una breve explicacion de como descargar las dependencias necesarias y como ejecutar los ejercicios de la practica 2.


## 📦 Instalacion + dependencias

Para ejecutar este proyecto es necesario tener instalado:

- Python 3
- Jupyter Notebook

Paso 1:
Descargar Python desde su sitio oficial:
[Python](https://www.python.org/)

Verificar instalacion.

```bash
  python --version
```
Paso 2:
Descargar Jupyter Notebook.
Luego de descargar Python, copiar y ejecutar este comando en la consola.

```bash
  pip install notebook
```


## ▶️ Cómo ejecutar el programa

1. Descargar el archivo del programa que quieras ejecutar (o clonar el repositorio directamente).

2. Abrir el archivo descargado en Visual Studio Code (u otro IDE, preferentemente VSC)

3. Debera instalar las siguientes extensiones en VSC (presionar el icono de tres cuadrados a la izquierda de la ventana):
- Python (de Microsoft).
- Jupyter (de Microsoft).

4. Ejecutar el notebook

- Abrir el archivo `.ipynb`
- Seleccionar el interprete de Python (arriba a la derecha)
- Hacer clic en "Run All" o ejecutar las celdas una por una

El programa se ejecutará directamente dentro de Visual Studio Code.
